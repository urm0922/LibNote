## LibNoteとは

LibNoteは、複雑な図書館業務に関するマニュアル・FAQ・対応ノウハウを一元管理し、どんな人でも、必要な情報にすばやくアクセスできるようにする業務支援アプリです。

## 制作背景

いくつも拠点を持つ大規模図書館では、正規職員・委託スタッフ・新人スタッフなど業務経験やITリテラシーが多様で、流動的な人員のもと同じサービス品質を求められます。サービスは日々複雑化しており、従来の紙やPDF中心のマニュアルは文量があり読みづらく、うまく活用されていないという課題がありました。

LibNoteは、こうした現場課題のもとに、業務ナレッジをわかりやすく整理・共有することを目的に作成しました。
