## LibNoteとは

LibNoteは、複雑な図書館業務に関するマニュアル・FAQ・対応ノウハウを一元管理し、どんな人でも、必要な情報にすばやくアクセスできるようにする業務支援アプリです。図書館に限らず、拠点を複数持つ現場スタッフの業務知識を標準化するアプリとして活用できます。


## 制作背景

いくつもの分館を持つ大規模図書館では、正規職員・委託スタッフ・新人スタッフなど業務経験やITリテラシーが多様で、流動的な人員のもと同じサービス品質を求められます。サービスは日々複雑化しており、従来の紙やPDF中心のマニュアルは文量があり読みづらく、うまく活用されていないという課題がありました。

LibNoteは、こうした現場課題のもとに、業務ナレッジをわかりやすく整理・共有することを目的に作成しました。


## 問題点
- 中央館と分館との情報格差があり、業務の情報が適切に共有されていない。
‐ 紙やPDFの業務マニュアルは、業務が複雑化して量が多く、読むのに抵抗がある。堅苦しい表現で難解である。
‐ 業務上の不明点は結果的に中央館スタッフに問い合わせることになり、組織全体の業務量が増加する。
- 図書館は基本的に土日祝日問わずサービス提供をしているので研修の機会が少ない。

## 解決策
- スタッフ間の業務情報のばらつきを減らし、問い合わせ対応、研修、マニュアル更新を一元化する
‐ 生成Aiで情報を整理してマニュアルを読みやすくする
‐ スタッフの問い合わせからナレッジ記事を作成し、業務情報を蓄積する
- ナレッジ記事から研修用クイズを自動生成する
